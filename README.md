# 使い方

1. `skininjector-v2.exe`を起動します。

2. 起動した後、置き換え後のスキンパックのフォルダーをウィンドウにドラッグアンドドロップするか、`スキンパックを選択`ボタンをクリックしてスキンパックのフォルダーを選択します。

3. 置き換えたいスキンパックをリストから選択します。

4. injectボタンをクリックします。

5. 完了です。

# 暗号化について

iosやアンドロイド向けの4Dスキンは10周年用スキンパックを置き換える前提で暗号化されている場合がります。その場合は暗号化のチェックを外してください。

もしうまくいかない場合は、一度スキンデータの削除をクリックして、マイクラのスキン変更画面を開いてからやり直してください。
